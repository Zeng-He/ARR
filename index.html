¿Qué es SARR?

El Sistema de Automatización de Registro de Reos(SARR) provee una administración sencilla de los recursos de cualquier institución carcelaria.

Ideado por el grupo de desarrollo Loading..., este sistema esta diseñado para integrar facilidades de software y hardware, implementado la lectura inteligente de huellas dactilares. Leer más
